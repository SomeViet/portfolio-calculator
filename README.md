## Run Locally

Clone the project

```bash
  git clone https://github.com/SomeViet/portfolio-task-list
```

Go to the project directory

```bash
  cd portfolio-task-list
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
